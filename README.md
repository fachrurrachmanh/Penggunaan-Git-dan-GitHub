# Penggunaan Git dan GitHub

#### Istilah penting dalam Git:
- **Repository**: Tempat penyimpanan project dan riwayat perubahannya
- **Commit**: Penyimpanan perubahan pada repository
- **Branch**: Cabang pengembangan
- **Merge**: Proses penggabungan branch
- **Remote**: Repository yang berada di server (GitHub)
